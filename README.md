caremad.io
==========

Installation
------------

1. Grab the source:

   ```bash
    $ git clone git@github.com:dstufft/caremad.io.git
   ```

2. Install the Python requirements:

   ```bash
    $ make install
   ```

3. Download the Fonts from Typography.com and put them in
   ``_themes/caremad/static/fonts``

4. Start the LiquidLuck server

   ```bash
    $ make serve
   ```
